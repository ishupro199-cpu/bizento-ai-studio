IMAGE GENERATION PIPELINE
Implement the following AI workflow.
Step 1 — Image Upload
User uploads a product image.
Store the original image in the image storage system.

Step 2 — Product Analysis
Use a vision model to analyze the uploaded image.
Extract product information such as:
product category
color
material
object type
Use this information to improve prompt generation.

Step 3 — Background Removal
Remove the background from the product image.
Output should be a transparent product image.
This step allows the system to place the product into new scenes.

Step 4 — Scene Generation
Generate a background scene based on:
selected style
selected tool
prompt text
Examples:
luxury marble scene
studio lighting
flower product scene
cinematic neon scene

Step 5 — Product Compositing
Place the extracted product image into the generated scene.
Adjust:
lighting
shadow
perspective
reflection
Ensure the final image looks realistic.

Step 6 — Image Rendering
Render the final output image.
Produce multiple variations.
Example:
3 generated images per request.

STEP 7 — SAVE OUTPUT
Save generated images into storage.
Store metadata including:
prompt
tool
style
model used
generation time
credits used
Images should automatically appear in:
My Catalogs
My Ads
Images
History

MODEL BEHAVIOR
The platform supports two models.
Nano Bana Flash
• faster generation
• lower compute cost
• ideal for catalog images
Nano Bana Pro
• higher realism
• better lighting and reflections
• ideal for ads and cinematic visuals
When Nano Bana Pro is selected, the generation pipeline should prioritize higher quality rendering.
