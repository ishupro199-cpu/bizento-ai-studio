## 🎯 Complete Landing Page Prompt — Pixalera

### Design System

- **Theme:** Dark mode only ("Obsidian Kiwi")
- **Background:** Deep night black `#0D0F14` (HSL: 220 30% 6%)
- **Primary Accent:** Electric Kiwi `#89E900` (HSL: 85 100% 46%)
- **Primary Foreground (text on kiwi buttons):** `#0D0F14`
- **Card Background:** `#12141A` | Border: `#1E2028`
- **Muted Foreground:** `#8A8F9E`
- **Typography:**
  - Headings: **Bricolage Grotesque** (Google Fonts), font-extrabold, tight tracking
  - Body: **Geist Sans** (system/loaded)
  - Code/Prompts: **Geist Mono** (monospace)
- **Glow Utilities:** Kiwi box-shadow glows (`0 0 40px hsl(85 100% 46% / 0.15)`), text-shadow glows
- **Grid Pattern:** Subtle `.bg-grid` using repeating SVG lines at low opacity
- **Border Radius:** `0.75rem` (12px) default

### Global Effects
- **Marquee animation:** `@keyframes marquee { 0% translateX(0) → 100% translateX(-50%) }` 15s linear infinite
- **Typing animation:** Character-by-character reveal at 35ms, 2.5s hold, then reset cycle
- **Kiwi pulse glow:** `@keyframes pulse-kiwi` alternating box-shadow intensity
- **Fade-in-up:** `translateY(30px) opacity:0 → translateY(0) opacity:1` on scroll
- **Hover glow classes:** `.glow-kiwi-sm`, `.glow-kiwi`, `.glow-kiwi-text`
